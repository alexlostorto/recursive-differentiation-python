# rdiff.py

## Description

A python program to recursively differentiate equations.

Feel free to clone and hack on it.

## Status

Half-baked, only use for fun.

## Example

Run program:

```sh
python rdiff.py
```

Equation input:

```
enter equation:
> 5x3 + 4x2 + 3x - 6
# means: 5x³ + 4x² + 3x - 6
```
